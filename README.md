# TM4C123 18-Bit TFT LCD Driver

An embedded graphics project demonstrating image rendering on a 240x400 TFT LCD using a TM4C123GH6PM microcontroller.

The LCD is connected through an 18-bit parallel interface implemented entirely using GPIO control through TivaWare.

Features:

- TFT LCD initialization
- 18-bit parallel communication
- RGB666 color output
- RGB888 image conversion
- LVGL image support
- Static image rendering
- Custom LCD driver implementation
  ### MCU

TM4C123GH6PM

- 80 MHz Cortex-M4F
- 256 KB Flash
- 32 KB SRAM
  ### DISPLAY
Resolution: 240x400

Interface: 18-bit Parallel

Color Mode: RGB666
